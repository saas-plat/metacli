# Usage
<!-- usage -->
```sh-session
$ npm install -g metacli
$ metacli COMMAND
running command...
$ metacli (-v|--version|version)
metacli/1.0.0 win32-x64 node-v10.15.3
$ metacli --help [COMMAND]
USAGE
  $ metacli COMMAND
...
```
